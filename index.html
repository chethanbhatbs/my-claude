What you get

at a glance.

Everything Claude quietly keeps on disk, organized into six views you can actually read.

KPIs, weekly spend trend, cache savings, busiest project, and an activity heatmap — the one screen you live on.

✦

Skills

Every skill by category, how often you actually invoke each one, unused-skill flags, and a readable SKILL.md viewer.

Install in under a minute

Run one command.
It does the rest.

No accounts, no config, no cloud. The installer sets up an isolated Python environment and opens the dashboard.

01

Paste one command

Run the install line in your terminal. It downloads a small self-contained app to ~/.my-claude.

02

It sets up locally

A one-time ~20s setup creates an isolated venv. No global installs, no admin rights, no dependencies to hunt.

03

Opens in your browser

Launches at 127.0.0.1:8766, reading your own ~/.claude. Bookmark it and re-run run.sh anytime.

Private by design

Your data never leaves your computer.

Your conversation transcripts are sensitive. This tool treats them that way.

  • ●Binds to 127.0.0.1 only — nobody else on your network can reach it.
  • ●Zero external calls. No telemetry, no analytics, no upload.
  • ●Reads ~/.claude on the machine it runs on. Nothing else.
  • ●Share the tool, not the data — each person sees only their own.
