Three steps, one command

Run it once. It does the rest.

No accounts, no config, no cloud. The installer sets up a local Python environment and opens the dashboard in your browser.

01

Paste one command

Run the install line in your terminal. It downloads a small self-contained app to ~/.my-claude.

02

It sets up locally

A one-time ~20s setup creates an isolated Python env. No global installs, no admin rights.

03

Opens in your browser

The dashboard launches at 127.0.0.1:8766, reading your own ~/.claude. That's it.

Private by design

Your data never leaves your computer.

Your conversation transcripts are sensitive. This tool treats them that way.

  • ●Binds to 127.0.0.1 only — nobody else on your network can reach it.
  • ●Zero external calls. No telemetry, no analytics, no upload.
  • ●Reads ~/.claude on the machine it runs on. Nothing else.
  • ●Share the tool, not the data — each person sees only their own.
